# 智控科技 - 工業自動化解決方案領導者

這是一個現代化的自動化科技公司官網專案，展示公司的產品、解決方案和成功案例。

## 專案結構

```
AI_CODE/
├── index.html      # 主頁面
├── styles.css      # 樣式檔
├── script.js       # JavaScript交互邏輯
├── README.md       # 專案說明
└── .gitignore      # Git忽略配置
```

## 技術棧

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome 6.x (CDN)

## 功能特性

- ✅ 回應式設計，支持移動端和桌面端
- ✅ 平滑滾動導航
- ✅ 案例輪播自動播放
- ✅ 產品卡片懸停動畫
- ✅ 聯繫表單交互
- ✅ 固定導航欄
- ✅ 現代化UI設計

## 快速開始

### 方式一：直接打開

```bash
# 使用流覽器打開 index.html
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

### 方式二：使用本地伺服器

```bash
# 使用 Python 啟動簡易伺服器
python -m http.server 8000

# 使用 Node.js http-server
npx http-server -p 8000
```

然後訪問 http://localhost:8000

## 部署到 GitHub Pages

1. 創建 GitHub 倉庫
2. 上傳專案檔
3. 在倉庫設置中啟用 GitHub Pages
4. 選擇 `main` 分支和根目錄

## 自定義內容

### 修改公司資訊

編輯 `index.html` 檔中的以下內容：
- 公司名稱和Logo
- 聯繫資訊（地址、電話、郵箱）
- 產品和解決方案描述
- 成功案例內容

### 修改主題顏色

編輯 `styles.css` 檔中的主色調變數：
- 主色：`#0066cc`
- 次色：`#00ccff`
